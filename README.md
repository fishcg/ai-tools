# ai-tools

Providing a web-based AI assistant through the openAI interface

## Environment requirements

golang 1.17 or above

## Installation steps

```sh
# build
go build

# 复制 config 并对其进行修改
cp config_bac.yml config.yml

# run
./ai-tools
```
